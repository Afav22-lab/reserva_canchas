📋 PRÓXIMOS PASOS PARA USAR EL SISTEMA:
════════════════════════════════════════════════════════════════

PASO 1: Crear un Superusuario (Administrador)
──────────────────────────────────────────────
Abre una terminal en la carpeta del proyecto y ejecuta:

   cd C:/Users/PIPE/Desktop/reserva_canchas
   .\venv\Scripts\activate
   python manage.py createsuperuser

Te pedirá:
   - Nombre de usuario: (elige uno, ej: admin)
   - Email: (opcional, puedes presionar Enter)
   - Contraseña: (elige una segura)
   - Confirmar contraseña


PASO 2: Iniciar el Servidor
──────────────────────────────────────────────
Opción A - Fácil (Recomendado):
   → Doble clic en el archivo "iniciar.bat"

Opción B - Manual:
   → Abre terminal y ejecuta:
     cd C:/Users/PIPE/Desktop/reserva_canchas
     .\venv\Scripts\activate
     python manage.py runserver


PASO 3: Crear Canchas (Como Administrador)
──────────────────────────────────────────────
1. Abre tu navegador en: http://127.0.0.1:8000/admin/
2. Inicia sesión con el superusuario que creaste
3. Haz clic en "Canchas" → "Agregar Cancha"
4. Llena los campos:
   - Nombre: Ej. "Cancha Fútbol 5"
   - Descripción: Ej. "Cancha sintética con iluminación"
   - Activa: ✓ (marca el checkbox)
5. Haz clic en "Guardar"
6. Crea varias canchas para probar el sistema


PASO 4: Probar el Sistema (Como Usuario)
──────────────────────────────────────────────
1. Abre: http://127.0.0.1:8000/
2. Haz clic en "Registrarse"
3. Crea una cuenta de usuario
4. Inicia sesión
5. Explora las canchas disponibles
6. Haz clic en "Reservar" en una cancha
7. Selecciona una fecha
8. Confirma la reserva
9. Ve a "Mis Reservas" para ver tu reserva

❓ SOLUCIÓN DE PROBLEMAS:
════════════════════════════════════════════════════════════════

Problema: "No module named django"
Solución: Activa el entorno virtual y ejecuta:
          pip install -r requirements.txt

Problema: "Port already in use"
Solución: Usa otro puerto:
          python manage.py runserver 8001

Problema: No aparecen las canchas
Solución: Asegúrate de haber creado canchas desde el admin
          y que estén marcadas como "activas"
