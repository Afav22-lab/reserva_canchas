════════════════════════════════════════════════════════════════
📊 FLUJO DE DATOS DEL SISTEMA
════════════════════════════════════════════════════════════════

1. USUARIO ACCEDE A UNA URL
   ↓
2. config/urls.py → Redirige a canchas/urls.py
   ↓
3. canchas/urls.py → Llama a la vista correspondiente
   ↓
4. canchas/views.py → Procesa la lógica
   ↓
5. canchas/models.py → Interactúa con la base de datos
   ↓
6. canchas/views.py → Prepara los datos
   ↓
7. templates/canchas/*.html → Renderiza la página
   ↓
8. USUARIO VE LA PÁGINA


════════════════════════════════════════════════════════════════
🔐 SISTEMA DE AUTENTICACIÓN
════════════════════════════════════════════════════════════════

Usuario No Autenticado:
├── Puede ver la página principal
├── Puede ver canchas disponibles (sin reservar)
├── Puede registrarse
└── Puede iniciar sesión

Usuario Autenticado:
├── Puede ver todas las canchas
├── Puede reservar canchas
├── Puede ver sus reservas
├── Puede cancelar sus reservas
└── Puede cerrar sesión

Administrador (Superusuario):
├── Todo lo anterior +
├── Acceso al panel admin (/admin/)
├── Crear/editar/eliminar canchas
├── Ver todas las reservas
└── Gestionar usuarios

════════════════════════════════════════════════════════════════
🔄 FLUJO DE RESERVA
════════════════════════════════════════════════════════════════

1. Usuario inicia sesión
   ↓
2. Ve lista de canchas disponibles
   ↓
3. Selecciona una cancha
   ↓
4. Elige fecha y agrega notas (opcional)
   ↓
5. Sistema valida:
   - Fecha no sea pasada
   - No exista reserva duplicada
   ↓
6. Crea la reserva con estado "confirmada"
   ↓
7. Muestra mensaje de éxito
   ↓
8. Usuario puede ver su reserva en "Mis Reservas"
   ↓
9. Usuario puede cancelar la reserva si lo desea

════════════════════════════════════════════════════════════════
🚀 COMANDOS DE DESARROLLO
════════════════════════════════════════════════════════════════

Iniciar proyecto:
  django-admin startproject config .

Crear app:
  python manage.py startapp canchas

Migraciones:
  python manage.py makemigrations
  python manage.py migrate

Superusuario:
  python manage.py createsuperuser

Servidor:
  python manage.py runserver

Shell interactivo:
  python manage.py shell

Verificar proyecto:
  python manage.py check
